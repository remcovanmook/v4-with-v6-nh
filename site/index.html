v4gwd

Host daemon for draft-vanmook-intarea-ipv6-resolved-gateway — IPv4 over IPv6-only segments, gateway 192.0.0.11 resolved from the IPv6 neighbor cache instead of ARP.

Architecture-independent packages (pure Python + pyroute2). Installing enables and starts v4gwd.service; it manages every ethernet interface and stays inert until a DHCPv4 server hands out 192.0.0.11.

Debian / Ubuntu

wget https://remcovanmook.github.io/v4-with-v6-nh/v4gwd.deb
sudo apt install ./v4gwd.deb

Fedora / RHEL

wget https://remcovanmook.github.io/v4-with-v6-nh/v4gwd.rpm
sudo dnf install ./v4gwd.rpm

Direct links follow the current release: v4gwd.deb · v4gwd.rpm. Versioned filenames and other host builds (macOS, FreeBSD, Windows, systemd-networkd) are on the releases page.
